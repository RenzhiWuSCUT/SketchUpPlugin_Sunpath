# SketchUp插件：太阳轨迹绘制

一个用于SketchUp的太阳轨迹绘制插件，可在模型中可视化太阳路径，辅助日照分析、遮阳研究与建筑设计。

---

## 功能

- 在SketchUp中绘制全年太阳轨迹图
- 通过设置对话框调整城市、日期及显示参数
- 生成单日太阳遮挡扇面图
- 用于快速分析日照条件与遮阳关系

---

本插件主要作为SketchUp中的可视化辅助工具，帮助用户在建模和设计过程中更直观地理解太阳路径及遮挡关系。
